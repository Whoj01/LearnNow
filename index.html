Somos uma ideia que visa trazer acessibilidade para os estudos de diversar pessoas

Com o LearnNow disponibilizamos aos alunos que por algum motivo não conseguem ir a escola ou moram em regiões afastadas a oportunidade de aprender completamente de graça.

Sobre nós

Confira algumas das funcionalidades que disponibilizamos no LearnNow.

Entrar como aluno

Na pagina de login do app, umas das opções de login é como aluno, quando se tem acesso por ele, é disponibilizado acesso as aulas e tarefas.

Entrar como professor

Outra opção de login é como professor, tendo acesso como professor você pode gravar aulas, dar aulas ao vivo, acessar a página de tarefas feitas e aplicar as notas.

Entrar como responsável

A ultima opção de login é como responsável, tendo acesso como responsável você pode monitorar o desempenho de quem é responsável e entrar em contato com professores.

Conheça nossos serviços

Confira Serviços que disponibilizamos para os usuários.

Disponibilizamos aos alunos isenção de uso de internet para pesquisas e assistir aulas, destribuindo chips com rede ilimitada

Contamos também com suporte a partir das 7 até as 23 horas para alunos, pais e professores, para não ter duvida nenhuma de como usar o LearnNow.

Caso o aluno trabalhe ou tenha perdido a aula, ele não precisa se preocupar, a aula ficará gravada em nossa plataforma para acessa-lá mas com uma data limite para evitar atrasos.

Para melhorar a comunicação entre os alunos, é criado uma sala de chat para que eles possam debater e organizar trabalhos, tudo com monitoramento do professor.

Para incentivar o aluno a continuar estudando e ver as aulas, a cada aula assistida é liberado a tarefa correspondente com ela, ficando assim disponivel para ser feita e avaliada pelo professor.
